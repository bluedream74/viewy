Viewy[ビューイー] プライバシーポリシー
ユーザーから提供される情報
  • 未登録ユーザー(ビジター)
    • 未登録ユーザーは登録せずに写真やビデオを閲覧できます。 収集および処理されるデータはすべて匿名化されます。 ウェブサイトが機能するために必要な必須情報のみが収集および処理されます。
  • 登録済みユーザー(ベーシック、パートナー)
    • ユーザー名、メールアドレス、パスワード、性別、生年月日(該当する場合)などのユーザー登録情報を当社に提供します
  • 投稿コンテンツ
    • あなたが自発的にオンラインで公開した個人情報やコンテンツ(どこで、誰によって作成されたかなどの関連メタデータを含む)は公開されます。
    • こうした情報は他のユーザーによって読み取られ、収集、使用、開示される可能性がありますが、当社は、ユーザーの投稿に誰がアクセスするか、またはユーザーが自発的に投稿または送信した情報に対して他のユーザーが何を行うかを制御することはできません。
  • Web ブラウザの情報
    • Web リクエスト、IP アドレス、ブラウザの種類、ブラウザの言語、参照 URL、プラットフォームの種類、ドメイン名、日付と時刻など、Web ブラウザが送信する特定の情報が当社のサーバーによって自動的に記録されます。
  • ユーザーの身元または年齢の証明
    • ユーザーがパートナーに加入するにあたり、本プラットフォームの利用年齢に達していることを確認するため、または確認が求められるその他の状況において、ユーザーに身元または年齢の証明の提示を求める場合があります。
  • 当社へ連絡する際に送られた情報
    • ユーザーがサポートまたはフィードバックのために当社に連絡する場合を含め、ユーザーが当社に送信する連絡に記載された情報。
  • 当社は、ユーザーのデータを使用して、直接的または間接的にユーザーを特定せず、ユーザーとは関係のない集約された洞察を作成および共有する場合があります。 このような集合情報は個人情報ではありません。
  • 当社の Web サイトは、18 歳未満、または Web サイトにアクセスする管轄区域で該当する成人年齢未満の人 (「未成年者」) を対象としたものではなく、未成年者が Web サイトを使用することは禁止されています。 当社は未成年者から故意に個人情報を収集しません。
自動的に収集される情報
  • 利用情報
    • ユーザーに表示されたコンテンツや広告、ユーザーの行動、検索履歴、いいねした投稿、フォローしたアカウント、利用時間帯、滞在時間等の情報を収集します
  • 推測情報
    • 当社は、コンテンツをパーソナライズするために、ユーザーの関心、性別、および年齢層などのユーザーの好みや属性を推測します。
  • クッキー
    • 当社、ならびに当社のサービスプロバイダおよびビジネスパートナーは、クッキーおよびその他の類似技術(ウェブビーコン、フラッシュクッキーなど)(総称して「クッキー」)を利用します。その目的は、自動的に情報を収集し、ユーザーがどのように本プラットフォームを利用するか(ユーザーが最も頻繁に閲覧するページやコンテンツに対するユーザーの行動を含みます)を測定·分析すること、本プラットフォームを利用するユーザーのエクスペリエンスを向上すること、本プラットフォームを改善すること、ユーザーに広告を表示すること、および広告やその他のコンテンツの効果測定を行うことです。また、当社および当社のパートナーは、他のプラットフォームやウェブサイト上で、本プラットフォームをプロモーションするためにもクッキーを使用します。クッキーにより、本プラットフォームは一定の機能を提供することが可能となります。ウェブビーコンは、「pixelタグ」または「クリアGIF」とも呼ばれる、非常に小さな画像または画像に埋め込まれた小さなデータであり、クッキー、ページの閲覧日時、pixelタグが配置されたページの説明、およびユーザーのコンピューターまたはデバイスからの類似の情報を認識することができます。特定のクッキーを無効にする方法については、以下の「ユーザーの権利と選択」のセクションをご覧ください。
利用目的

  • 広告、宣伝、マーケティングのため

  • 独立したリサーチャー
    • 当社は、一定の基準を満たすリサーチを促進するため、独立したリサーチャーとユーザーの情報を共有します。
  • 法的な理由による共有
    • 当社は、法的に要求される場合、または以下の目的のために合理的に必要な場合、法執行機関、公的機関またはその他の機関とユーザーの情報を共有します。
      • 法的義務、手続き、または要請を遵守するため。
      • 当社のサービス規約、ならびにその他の契約、ポリシー、および基準を執行するため(これらの潜在的違反の調査を含みます)。
      • セキュリティ上の問題、詐欺的行為、または技術的な問題の発見、防止またはその他の方法による対処のため。
      • 法により要求または認められた、当社、当社のユーザー、第三者または一般の人々の権利、財産または安全を保護するため(詐欺的行為の防止およびクレジットリスクの軽減を目的とした他の企業や組織との情報交換を含みます)。
  • 企業買収·合併またはその他の事業取引
    • 当社は、継続企業として、または当社の資産の一部として、合併、買収、売却または譲渡が行われた場合(清算、破産等による場合も含みます)に、当社が収集した個人情報のすべてのカテゴリーを買い手またはその他の後継者に開示することがあります。

  • ユーザーが希望する場合、ユーザーは設定画面で自身のアカウントを完全に削除することができます。
  • ユーザーは個人データの取り扱いに関し、当社の管理者または監督機関に不服を申し立てることができます。ユーザーは前項各号の権利を自由に行使することができ、当社はユーザーによる権利の行使を理由として商品またはサービスの提供を拒否することはありません。これらの権利について問い合わせるまたは権利の行使を希望される場合、当社にご連絡ください。なお、ユーザーの権利は絶対的なものではなく、適用法令において認められる根拠がある場合には、ユーザーからの請求にお応えしかねることがございますので、あらかじめご了承ください。

個人情報の保管
  • 当サイトはユーザーのアカウントが有効である限り情報を保持します。また契約上および法律上の義務を遵守するために必要な場合、正当なビジネス上の利益がある場合(本プラットフォームの改善および開発、ならびにその安全性、セキュリティおよび安定性の向上など)、ならびに訴訟に関する遂行または防御のためにも、情報を保持します。
  • ユーザーが当社のサービス規約、コミュニティガイドライン、またはその他の条件やポリシーに違反した場合、当社はユーザーのプロフィールおよびユーザーコンテンツを直ちに一般公開から削除することができますが、違反を処理するためにユーザーに関するその他の情報を保有する場合があります。
  • このプライバシー ポリシーに定められた目的のためにユーザーの個人情報を処理する必要がなくなった場合、当社はユーザーの個人情報を当社のシステムから削除します。
  • 可能な場合、上記の「ユーザーの権利」セクションで説明したように、当社はユーザーの要求に応じてユーザーの個人情報も削除します。
